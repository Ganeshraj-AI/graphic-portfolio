# Portfolio Philosophy

This website is not a resume.

It is an interactive digital exhibition called **The Graphic Archive**.

The website itself should feel like a graphic design project.

Every project is an exhibition piece.

The artwork must always be the hero.

---

# Current Collection

I currently have only a small number of projects.

Do NOT duplicate projects.

---

# Typography & Design Direction

Use strong editorial typography.

Mix serif, sans-serif, and monospace fonts.

Create bold visual hierarchy.

The website should feel like a premium design exhibition rather than a corporate portfolio.

Decorative elements such as print marks, crop marks, tape, registration marks, and metadata should support the artwork, never overpower it.

---

# Content Rules

Write content only from available information.

Do not invent:

- clients
- awards
- years
- project goals
- project descriptions
- tools used
- achievements

If information is unavailable, keep it minimal.

Authenticity is more important than filling empty space.

# Final Goal

Create a timeless portfolio that grows naturally as I add more work.

Every future image added to `assets/posters` should automatically become part of the archive.

The portfolio should feel like visiting a curated graphic design exhibition.
